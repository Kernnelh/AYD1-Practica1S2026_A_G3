# Frontend (React) — Vistas Registro / Login / Perfil

Instalación y ejecución rápida:

```bash
npm install
npm run dev
```

Notas:
- Las rutas de la API son placeholders: `http://localhost:3000/api/*`. Ajusta `src/api.js` según tu backend.
- Vistas creadas: `src/pages/Register.jsx`, `src/pages/Login.jsx`, `src/pages/Profile.jsx`.
